# The_NextNode-AI

AI-powered Career Guidance System built using FastAPI and Azure AI Foundry.

## Features

* Resume Upload (PDF)
* Resume Text Extraction
* AI-Powered Career Analysis
* Career Readiness Score
* Skill Gap Identification
* Certification Recommendations
* Learning Roadmap
* Interview Preparation Guidance

## Tech Stack

* Python
* FastAPI
* Azure AI Foundry
* Azure AI Agent
* OpenAI SDK
* pdfplumber

## Project Structure

frontend/

## How It Works

1. Upload Resume PDF
2. Extract Resume Text
3. Send Resume to Azure AI Agent
4. Generate Career Report
5. Display Recommendations
